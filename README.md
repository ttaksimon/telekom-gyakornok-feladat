# NetConf szerver
Ezt a feladatot egy gyakornoki kiválasztási folyamat részeként valósítottam meg, **Python** nyelven.

## Fejlesztői környezet
- **IDE:** Visual Studio Code (VS Code)
- **Környezet:** Python virtuális környezet (venv)
- **Operációs rendszer:** macOS

## Tesztelés
A funkcionális teszteket Cisco sandboxokban végeztem.
- Elsődleges eszköz: **Catalyst 8000 Always-On Sandbox**
- Előnye: Nem igényelt VPN-kapcsolatot, így gyorsabbá tette a fejlesztést.
